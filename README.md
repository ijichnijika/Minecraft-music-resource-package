# §6孤独摇滚 Minecraft 音乐资源包

一个自定义的 Minecraft 音乐资源包，替换游戏中的背景音乐。

## 📝 简介

本资源包替换了 Minecraft 原版的背景音乐，提供了全新的游戏体验。包含了主世界、下界、末地等各个维度的音乐。

## 🎵 包含内容

- **主世界音乐** - 游戏主要场景的背景音乐
- **下界音乐** - 下界各个生物群系的音乐
  - 下界荒地 (Nether Wastes)
  - 绯红森林 (Crimson Forest)
  - 灵魂沙峡谷 (Soul Sand Valley)
- **末地音乐** - 末地维度音乐
- **沼泽音乐** - 沼泽生物群系音乐
- **水下音乐** - 水下场景音乐
- **菜单音乐** - 主菜单界面音乐

## 💿 安装方法

1. 下载本资源包
2. 将资源包文件放入 Minecraft 的 `resourcepacks` 文件夹
   - Windows: `%appdata%\.minecraft\resourcepacks`
   - macOS: `~/Library/Application Support/minecraft/resourcepacks`
   - Linux: `~/.minecraft/resourcepacks`
3. 启动 Minecraft，进入"选项" → "资源包"
4. 选择本资源包，点击"完成"
5. 装载后即可享受新的背景音乐！

## ⚙️ 兼容性

- **资源包格式**: 15
- **兼容版本**: Minecraft 1.20+

## 📌 注意事项

> **重要**: 装载后才有音乐哦！请确保在游戏中正确加载了资源包。
